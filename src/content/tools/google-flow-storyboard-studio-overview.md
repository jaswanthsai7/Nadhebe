# Google Flow Storyboard Studio: Product Overview and Toolset

Google Flow's **Storyboard Studio** offers a specialized toolset designed to accelerate creative pre-production. This article reviews the core features and styles available.

## Core Features

1. **AI Script Segmenter**: Automatically parses and divides text into structured scenes.
2. **Visual Style Presets**: Allows creators to apply distinct presets, such as 3D animation, hand-drawn sketch styles, and photographic renders.
3. **Asset Manager**: Catalogs characters, props, and location tags to maintain stylistic consistency across frames.

The platform provides a solid framework for creators seeking to build structured, visually consistent storyboards directly from raw scripts.
